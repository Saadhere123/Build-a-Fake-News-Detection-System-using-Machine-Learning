                                                  Fake News Detection System 

Objective
This project detects whether a news article is Fake or Real using Machine Learning.

Technologies Used
- Python
- pandas
- scikit-learn
- nltk
- joblib

Features
- Text preprocessing (stopwords, lemmatization)
- TF-IDF feature extraction
- Logistic Regression model
- Accuracy, Precision, Recall, F1-score
- Confusion Matrix
- Real-time user input prediction
- Confidence score
- Model saving

                                                        How to Run

1. Train Model


2. Predict News


Dataset
Custom dataset with 200 samples (Fake & Real)
